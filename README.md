# AEB Tools

Public install repository for the **AEBTools.extension** pyRevit extension for Autodesk Revit.

This repository is made for **installation and use**, not for development.  
Users can download it, connect it in pyRevit, and start using the tools in Revit.

---

## What is this?

AEB Tools is a pyRevit extension created to provide simple custom tools for Revit users.

This public repository is the install location for users.  
Tools will be added and improved step by step over time.

---

## Current Tool

### Door Parameter From Room Number
Updates a selected writable door parameter using the related room number.

---

## Supported Revit Versions

Tested from **Revit 2022 to Revit 2027**.

If you face any issue in any version, please contact the **BIM Manager**.

---

## Installation

Follow these simple steps:

1. Download or clone this repository.
2. Keep the folder **AEBTools.extension** as it is.
3. Copy the repository folder to any proper location on your computer.
4. Open **Revit**.
5. Open **pyRevit Settings**.
6. Add the extension path by selecting the repository root folder or the **AEBTools.extension** folder.
7. Reload pyRevit.
8. Open Revit and confirm that the **AEB Tools** tab appears.

---

## Update Process

When a new version is released:

1. Download the latest version from this repository.
2. Replace the old repository content with the new one.
3. Reload pyRevit.
4. Check the tool in a test model before using it in a live project.
